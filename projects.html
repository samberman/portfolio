Food Roulette

A web app allowing a user to 'roll the dice' to get a restaurant recommendation based on their current location, the type of food they want to eat, and the radius they want to travel in meters. Built using Javascript, jQuery, the Yelp API and Ruby on Rails.

Javascript Jukebox (currently in development)

This static html page allows a user to play and pause a list of preloaded songs, skip to the next song, as well as add a song to the playlist via a form including an mp3 url. Built using HTML, CSS, Javascript, and JQuery.
